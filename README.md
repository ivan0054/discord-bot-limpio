Proyecto Maven Java - Discord Bot (esqueleto)
Contenido:
- pom.xml (configurado para Java 17)
- src/main/java/com/ejemplo/bot/... (clases principales)
- src/main/resources/config.properties (template)

Instrucciones:
1. Descarga y descomprime el ZIP.
2. Abre el proyecto en IntelliJ -> Import Project -> selecciona pom.xml.
3. Rellena src/main/resources/config.properties con tu token y API keys.
4. Run -> Maven -> package o ejecuta la clase com.ejemplo.bot.Main.
